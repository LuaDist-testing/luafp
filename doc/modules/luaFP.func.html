curry (num_args, func)
Takes your function arity (number of parameters) and a function, and turns it into a curried function. NOTE: We do not handle scope for you.

Parameters:

  • num_args how many parameters your function takes (the arity). This assumes no default parameters as those don't really work that well with curried functions.
  • func The function you wish to turn into a curried function.

Returns:

    curriedFunk Your curried function.

Usage:

    local func = require 'lua-fp'.func
function add(a, b) return a + b end
addCurry = func.curry(2, add)
add1 = addCurry(1)
print(add1(2)) -- 3
negate (func)
Takes a predicate function and reverse the value returned. If the function returns true, it'll now return false. If the function returns false, it'll now return true.

Parameters:

  • func Your predicate function, a function that only returns true or false. Parameter count doesn't matter, we'll unpack and call it for you.

Returns:

    negated Your reversed predicate function.

Usage:

    local func = require 'lua-fp'.negate
function isCow(o) return o == 'cow' end
print(isCow('cow')) -- true
isNotCowLulz = func.negate(isCow)
print(isNotCowLulz('cow')) -- false
print(isNotCowLulz('chicken')) -- true
